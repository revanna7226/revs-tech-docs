Repository

Local Repository

  1. Setup Local Repository

    helm repo index chartsrepo/

helm package revschart -d ../chartsrepo/

  2. Host Repository

  • You can host the chartsrepo/ directory on any web server or cloud storage that serves static files.

  • For example, you can use GitHub Pages, Amazon S3, or any web hosting service.

    # Example Local Hosting with Python HTTP Server cd chartsrepo/ python3 -m http.server –bind 127.0.0.1 8080 –directory . # Now your repository is accessible at http://127.0.0.1:8080/

  1. Add Repository to Helm

    helm repo add revsrepo http://127.0.0.1:8080/

  2. Update Repository

    helm repo update # Regenerate index if you add new charts helm repo index chartsrepo/

  3. Install Chart from Repository

    helm install revschart revsrepo/revschart

  4. Verify Installation

    helm list

  5. Helm Search Repository

    helm search repo revschart

GitHub Repository

  1. Setup GitHub Repository

    • Create a new GitHub repository (e.g., gitchartrepo).

    • Clone the repository locally.

    git clone https://github.com/revanna7226/gitchartrepo.git

  2. Go to the cloned directory

    cd gitchartrepo/

helm create demochart
helm package demochart -d .
helm repo index .

  3. Push to GitHub

    git add .
git commit -m "Add Helm chart and index"
git push origin main
  4. Host GitHub Pages

    • Go to the repository settings on GitHub.

    • Under the “Pages” section, set the source to the main branch and the root directory (/).

    • Save the settings. Your repository will be available at https://<your-username>.github.io/gitchartrepo/.

  5. Add GitHub Repository to Helm

    helm repo add gitrepo https://revanna7226.github.io/gitchartrepo/ helm repo list helm search repo demochart helm install gitapp gitrepo/demochart

  6. Verify Installation

    helm list

  1. OCI Registries

    1. Push Chart to OCI Registry

      helm chart save revschart oci://my-oci-registry/revscharts:0.1.0 helm chart push oci://my-oci-registry/revscharts:0.1.0

    2. Pull Chart from OCI Registry

      helm chart pull oci://my-oci-registry/revscharts:0.1.0 helm chart export oci://my-oci-registry/revscharts:0.1.0 –destination ./charts

    3. Install Chart from OCI Registry

      helm install ociapp oci://my-oci-registry/revscharts –version 0.1.0

    4. Verify Installation

      helm list

# . Clean Up

  1. Uninstall Helm Releases

    helm uninstall revschart helm uninstall gitapp helm uninstall ociapp

  2. Remove Repositories

    helm repo remove revsrepo helm repo remove gitrepo helm repo remove ocirepo

  3. Delete Local Files

    rm -rf chartsrepo/ rm -rf gitchartrepo/ rm -rf charts/

  1. Summary

    • You have successfully created and managed Helm chart repositories using local hosting, GitHub Pages, and OCI registries.

    • You can now package, distribute, and deploy Helm charts efficiently using these methods.
