MySQL Command Line Client

  1. Login to MYSQL using the command line client:

mysql -u root -p

Enter the password when prompted.

  1. To create a new database:

CREATE DATABASE my_database;

  1. To List all databases:

SHOW DATABASES;

  1. To use a specific database:

USE my_database;

  1. To list all tables in the current database:

SHOW TABLES;
