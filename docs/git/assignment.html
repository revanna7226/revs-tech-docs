Git Assignments

Git Basics Assignment - Your Instructions

  1. Create a new folder and initialize the repository

mkdir test
cd test
git init

  1. Paste the “instructions.txt” file into this folder

  2. Add a .txt file named “file-1” containing any text of your choice to the working directory

  3. Create a second .txt file named “file-2”

  4. Add “file-1” and “file-2” to the staging area - don’t add “instructions.txt”

  5. Change the initial text you added to “file-1”

  6. Now add all working directory files to the staging area

  7. Create the first commit

  8. Create a second branch named “feature” (two commands are possible)

  9. Add a third .txt file (“file-3.txt”) to this branch

  10. Create a new commit

  11. Add the following text to “file-3”: “I will be deleted”

  12. Add the updated file to the staging area

  13. Undo the staged change

  14. Add the following text: “Please add me to the master/main branch”

  15. Commit this latest change

  16. Merge the “master” (or “main”) branch with “feature”

  17. Delete the “feature” branch

And most importantly: Have fun with the assignment :)
